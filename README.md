## Daggerheart - Deutsch
Übersetzungsmodul für das Daggerheart System für Foundry VTT.

Anleitung
- installiere das Daggerheart-System von Foundryborne
- installiere die deutsche Übersetzung des Foundry Cores [core-translation-it-foundryVTT](https://foundryvtt.com/packages/core-translation-it-foundryVTT)
- installiere dieses Modul
- aktiviere die Module im Control Panel
- ändere die Sprache im Control Panel auf Deutsch.
